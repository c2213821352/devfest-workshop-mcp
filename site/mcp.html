什么是 MCP？

MCP（Model Context Protocol）是 Chrome DevTools 使用的协议之一，允许前端工具与浏览器或被调试目标进行 JSON 消息通信，用于调试、性能检测和页面控制。

Workshop 实现要点

  • 启动一个 WebSocket 服务（例如本 Workshop 的 MCP 服务器），接受 DevTools 前端连接。
  • 实现必要的命令，如 Performance.enable / Performance.getMetrics，用于采集性能数据。
  • 可借助 Protocol Monitor（DevTools 实验功能）观察消息往返过程。
